A lightweight plugin that adds a floating WhatsApp chat icon to your WordPress site for easy communication with visitors.

== Description ==

Social Chat Icon is a lightweight WordPress plugin that adds a floating WhatsApp chat icon to your website. It provides an easy way for your visitors to contact you directly through WhatsApp messaging.

= Features =
* Easy setup with WhatsApp number
* Customizable icon position (left/right)
* Adjustable icon size
* Custom default message
* Option to display on all pages or homepage only
* Clean and modern design
* Mobile-friendly
* Lightweight and fast
* Translation-ready
* Proper WordPress coding standards
* Secure input handling

= Recent Updates =
* Improved security and sanitization
* Enhanced CSS with better mobile support
* Added translation support (POT file included)
* Improved settings page UI
* Added success/error messages
* Better code organization
* Fixed URL handling
* Added proper activation/deactivation hooks

= Usage =
1. Install and activate the plugin
2. Go to Settings > Social Chat Icon
3. Enter your WhatsApp number with country code
4. Customize the icon settings
5. Save changes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/social-chat-icon` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings under Settings > Social Chat Icon

== Frequently Asked Questions ==

= How do I set up my WhatsApp number? =
Enter your full WhatsApp number including the country code without any spaces or special characters (e.g., 14155552671 for US number +1 415 555 2671).

= Can I customize where the icon appears? =
Yes, you can choose between left and right positions and control whether it appears on all pages or just the homepage.

= Is the plugin GDPR compliant? =
Yes, the plugin doesn't collect or store any visitor data.

= Is the plugin secure? =
Yes, the plugin follows WordPress security best practices including proper data sanitization and validation.
